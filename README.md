# ShopSphere

A modern e-commerce web application built using Flask and MySQL.

## Features

- User Authentication
- Admin Dashboard
- Product Management
- Shopping Cart
- Razorpay Payment
- Orders
- Product Reviews
- Search

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
- Razorpay

## Installation

```bash
pip install -r requirements.txt
python app.py
